個人簡介

大學就讀資訊管理系，在畢業之際我對於兩個領域有興趣，電子商務行銷領域跟軟體工程領域，最後決定先選擇電子商務行銷企劃作為我職場的開端，此開端其實也打開了我朝第二個興趣邁向的原因。工作後在業務上常有機會跨部門與工程師開會討論品牌網站的功能建置，而在我的工作上時會處理許多 excel 報表，我希望可以讓報表的工作簡單化，所以我請教了工程師前輩，得知python中的套件可以進行資料處理，所以就開始自學，當時也激發起自己想轉職從事軟體工程師的想法，從而慢慢的開始自學程式包含:在後台使用前端語法設定網站前台畫面、接觸到python因為想讓工作的報表簡單化，最後因公司業務縮編的關係而被資遣，但我不因此而挫敗，因為我認為危機就是個轉機，下定決心給自己一個機會轉職成工程師，希望可以加入Wehelp訓練營進行系統性的學習，獲得專業技術轉職軟體工程師。

    以台灣來說的話KKBOX、Yahoo、趨勢科技、IBM這些外商都是我一直以來所嚮往的公司，過往的經驗外商、台商都有待過，認為外商公司在工作環境、公司整體的氛圍、以及薪水福利都比起台商還好上許多，以我過往的工作經驗所積累的多為溝通力、平台運營能力等相對較軟的實力，想加入這類的公司難度相對較高開缺也比較少，往往這些公司的工程師開缺佔比大多數，我深知如果想要加入這樣的公司最直接的方式就是藉由提升自己的硬實力技能，提高競爭力。

    05

    如何克服困難？

    剛加入公司一個專營運動補給食品的新創部門時，主管讓我寫一個用於品牌對外拓展商務的demo簡報，需要完整並詳細闡述品牌核心理念及所有關於產品的資訊，作為對外商務開發使用。但當時身為公司新人的我對於產品中很多細部製程技術及營養品知識還不是很熟悉，我開始是按照產品既有廣告文宣撰寫，與主管對焦後的結果是簡報中的邏輯及核心理念的描述未相互串聯，檢討過後我認為是自身對於產品各項營養添加物的功能還沒有全面的了解，後續我主動向商品開發的營養師請教，把每個我不熟的營養添加物功能理解熟悉，並串連開會中主管所說明的品牌核心理念後，最終製作了一個獲得主管的稱讚與認同並指派我後續負責品牌商開。我認為遇到棘手問題時保持良好的心態，並冷靜思考才是最佳的處理方式。

    開發這份網頁時的技術心得

    這份網頁整體的製作思路，使用＜/center＞標籤實現整體頁面居中，利用＜hr/＞標籤製作水平線，進而通過屬性設置，設置水平線的顏色、寬度等，利用＜/a＞標籤製作錨點，實現從頁面的底部直接回到當前頁面的頂端。讓我印象深刻的部分是我在的程式碼中，把 css 獨立成一個樣式檔案，再利用＜link/＞將其引入 html 中，並套用 Selector 選擇器後改動同類型物件的樣式時也比較方便，程式碼也比較乾淨。
